---
name: hello-world-5-languages
description: Generate and return "Hello, world!" in 5 random languages. Use when the user asks for hello-world in multiple languages, random language greetings, or a quick multilingual demo.
---

Run `scripts/say_hello_5.py` and return its output exactly.

## Command

```bash
python3 scripts/say_hello_5.py
```

## Output format

Return 5 lines in this format:

`<Language>: <Hello, world! translation>`

Do not add extra commentary unless the user asks.
